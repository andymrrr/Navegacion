# Tecnologia

- React Native
- TypeScripts

# Descripcion

En este proyecto se trabaja las diferentes navegaciones los cuales son

- Navegacion Drawer (Principal)
- Navegacion Stack
- Navegacion Top Taps

# Ambiente de desarrollo

Para subir el proyecto debe seguir los siguientes pasos

- Clonar el Repositorio.
- Hacer un `npm i`
- Correr el aplicativo `npm start`
- Selecionar el dispositivo Deseado
